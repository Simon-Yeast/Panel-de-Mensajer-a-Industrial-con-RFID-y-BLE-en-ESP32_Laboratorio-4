# Panel-de-Mensajer-a-Industrial-con-RFID-y-BLE-en-ESP32_Laboratorio-4
Desarrollo de un panel de mensajería industrial basado en ESP32 con autenticación RFID, comunicación Bluetooth Low Energy (BLE), visualización en LCD y registro horario mediante RTC DS1307. Implementa control de acceso, manejo de sesiones y comunicación inalámbrica segura.
